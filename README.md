# Chess Python

# Game Features

- **Multiple Themes:** Press the 't' key to switch themes.

- **Valid Move Highlighting**

- **Last Move Indicator**

- **Pawn Promotion**

- **Castling**

- **En Passant**

- **Game Restart**: Press the 'r' key to restart game.
